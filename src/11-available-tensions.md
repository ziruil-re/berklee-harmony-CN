# 第 11 章 可用延伸音

## 可用延伸音表 (Available Tensions Chart)

| 和弦类型 | 可用延伸音 | 与调内关系 | 大九度规则的例外 |
|---------|-----------|-----------|---------------|
| maj7 | 9、13、♯11 | | ♯11 需标注于和弦符号中 |
| -7 | 9、11 | | 13 仅在 Dorian 调式语境中可用 |
| 7（属七） | 9、♯11、13 | | ♭9、♯9、♭5（见注 1）、♭13 |
| 7(sus4) | 9、11（作和弦音）、13 | | 极少数情况：♭9、♯9、♭13 |
| -7(♭5) | | 所有可用延伸音必须是和弦音上方大九度且调内音 | |
| dim（三和弦） | | 同上 | |
| dim7 | | （见注 2） | |
| -(maj7) | 9、11、13 | | maj7 作为和弦音 |
| maj6 | 9、♯11 | | maj7 作为和弦音 |
| -6 | 9、11 | | maj7 作为和弦音 |

> **注 1**：♭5 是一种特殊的延伸音情形，涉及五度音的变化。
>
> **注 2**：减七和弦的可用延伸音不使用 9、11、13 的编号方式。与其他七和弦不同，和弦音上方的大九度或小九度各产生四个可能的延伸音（而非三个），因此传统的编号系统不适用。减七和弦的具体可用延伸音将在和弦出现于具体调性语境时说明。

## 可用延伸音总结 (Summary of Available Tensions)

| 延伸音 | 可用于 |
|--------|--------|
| maj7 | maj7、-(maj7)、maj6、-6 |
| 9 | 所有和弦（当调内音时） |
| ♭9 和/或 ♯9 | 属七；极少数情况下用于 7(sus4) |
| 11 | 所有形式的小和弦；作为和弦音用于 7(sus4) |
| ♯11 | maj7 和 maj6（当调内音时）；属七；+7 |
| ♭13 | -7(♭5)；属七 |
| 13 | maj7；-(maj7)；属七；7(sus4) |

## 和弦符号中延伸音的标注

延伸音标注在和弦符号中分为两类：

1. **非自然音阶内的延伸音**——**必须**标注在和弦符号中。
2. **自然音阶内的延伸音**——可选择性地"提示"标注。

非调内延伸音**必须**写入和弦符号：

> 例如：Cmaj7(♯11)

在 C 大调中，Cmaj7 通常只有 9 和 13 可用。Cmaj7(♯11) 则明确指示使用了一个非调内延伸音。
