# Berklee 和声学 1 & 2（中文翻译）

本书是 Berklee College of Music（伯克利音乐学院）**Harmony 1** 和 **Harmony 2** 课程教材的全书中文翻译。

原书由 **Barrie Nettles** 编写，是伯克利和声学课程的经典教材。

## Harmony 1 内容概览

- **记谱基础**：五线谱、谱号、加线、变音记号
- **音高体系**：半音与全音、等音、音阶与调式
- **调号**：升号调与降号调的识别
- **音程**：种类、性质、转位
- **和弦**：三和弦与七和弦的构造、转位
- **自然音阶和声**：自然音阶和弦、延伸音、可用延伸音
- **和声功能**：Sus4 和弦、和弦音阶、重配和声、终止式
- **作业练习**：配套的书面练习题

## Harmony 2 内容概览

- **副属和弦**：副属和弦的定义、解决与可用延伸音
- **和声节奏**：强弱拍、和弦的节拍分布
- **扩展属和弦**：扩展属和弦链、欺骗解决
- **旋律分析**：动机、接近音、经过音、先现音、挂留音
- **声部进行**：和声连续性、基本和弦音、导音线
- **小调和声**：自然小调、和声小调、旋律小调
- **小调中的副属和弦**：线性套式
- **调式互换**：平行调式和弦的借用
- **关系 II-7 和弦**：II-V 进行与双重功能
- **布鲁斯**：布鲁斯音阶、12 小节形式、进行变体

## 翻译说明

- 全部正文及作业已逐句翻译为中文
- 音乐术语采用 **中文（English）** 格式，如"五线谱 (staff)"
- 和弦符号保留英文原样：Cmaj7、V7、II-7 等
- 乐谱示例以原书截图形式嵌入
