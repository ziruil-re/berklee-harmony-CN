# Berklee 和声学 1（中文翻译）

本书是 Berklee College of Music（伯克利音乐学院）**Harmony 1** 课程教材的全书中文翻译。

原书由 **Barrie Nettles** 编写，是伯克利和声学课程的经典入门教材，涵盖了从基础记谱法、音阶、音程，到和弦构造、自然音阶和声、延伸音与终止式等核心主题。

## 内容概览

- **记谱基础**：五线谱、谱号、加线、变音记号
- **音高体系**：半音与全音、等音、音阶与调式
- **调号**：升号调与降号调的识别
- **音程**：种类、性质、转位
- **和弦**：三和弦与七和弦的构造、转位
- **自然音阶和声**：自然音阶和弦、延伸音、可用延伸音
- **和声功能**：Sus4 和弦、和弦音阶、重配和声、终止式
- **作业练习**：配套的书面练习题

## 翻译说明

- 全部正文及作业已逐句翻译为中文
- 音乐术语采用 **中文（English）** 格式，如"五线谱 (staff)"
- 和弦符号保留英文原样：Cmaj7、V7、II-7 等
- 乐谱示例以原书截图形式嵌入
